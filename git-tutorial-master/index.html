Aprendendo a linha de comando

por Tuanny Ruiz

  • ls - lista o que tem dentro do diretório atual (List);
  • pwd - imprime o endereço do diretório atual (Print Working Directory);
  • cd argumento - muda o diretório (Change Directory);
  • cd .. - volta um diretório;
  • cd ~ - volta para o diretório raiz;
  • mkdir - cria um diretório onde você está (Make Directory);
  • touch - cria um arquivo no diretório atual;
  • cp argumento1 argumento2 - copia arquivos ou diretórios (Copy);
  • mv argumento1 argumento2 - move arquivos ou diretórios (Move);
  • rm - deleta arquivos e diretórios (Remove);
  • notepad nome-do-arquivo.txt - abre o arquivo no bloco de notas;
  • man - mostrar o manual;
  • whoami - identifica o usuário que está mexendo no sistema.
  • ls -a - lista os arquivos que começam com (.) e ficam escondidos;
  • ls -t - lista os arquivos por ordem de última modificação;
  • rm -r - remove um diretório e tudo que tem dentro dele;
  • git init - avisa o git para ficar de olho na pasta;
  • git status - mostra o que foi modificado, adicionado ou deletado da pasta;
  • git add nome-do-arquivo.html - sobe um arquivo específico;
  • git add . - sobe todos os arquivos;
  • git commit -m “inclui botão vermelho” - coloca uma etiqueta no que está subindo;
  • git remote add origin link - vincula o repositório do github com os arquivos no seu computador;
  • git log -mostra a timeline do nosso projeto com todos os commits feitos;
